### Source Code

#### Main Application
- `src/index.ts` - Express server entry point

#### Routes
- `src/routes/api.ts` - API endpoint handlers
  - POST /api/analyze - Analyze webpage with Gemini
  - POST /api/generate - Generate content
  - POST /api/extract - Extract data with selectors
  - GET /health - Health check

#### Services
- `src/services/gemini.service.ts` - Gemini AI wrapper
  - `generateContent(prompt, context)` - Text generation
  - `analyzePageContent(content, prompt)` - Content analysis

- `src/services/puppeteer.service.ts` - Puppeteer wrapper
  - `fetchPageContent(url)` - Fetch page text
  - `takeScreenshot(url, path)` - Capture screenshots
  - `extractData(url, selector)` - Extract with selectors

#### Utilities
- `src/utils/middleware.ts` - Express middleware
  - Request logging
  - Error handling

- `src/utils/helpers.ts` - Helper functions
  - Logger utility
  - URL validation
  - Text sanitization
  - Retry mechanism
  - Error formatting

#### Types
- `src/types/index.ts` - TypeScript interfaces
  - AnalyzeRequest/Response
  - GenerateRequest/Response
  - ExtractRequest/Response
  - HealthResponse
  - ErrorResponse

#### MCP Server
- `src/mcp-server/server.ts` - MCP server implementation
  - Tool: fetch_page_content

   - NODE_ENV - Environment (development/production)
   - GEMINI_API_KEY - Your API key
   - MCP_SERVER_PORT - MCP server port (default: 3001)
